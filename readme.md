# TaskFlow Backend

A Spring Boot backend application for managing tasks and task lists with support for priorities and statuses. This project uses PostgreSQL as the database and can be run locally with Docker Compose.

---

## Features

- Create, update, delete tasks and task lists  
- Set task priority: High, Medium, Low  
- Set task status: Open, Closed  
- Group tasks under task lists  
- RESTful API endpoints for all operations  

---

## Tech Stack

- Java 17 / Spring Boot  
- Spring Data JPA  
- PostgreSQL  
- Docker & Docker Compose  

---
